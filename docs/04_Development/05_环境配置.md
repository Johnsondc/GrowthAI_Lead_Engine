# 环境配置

## Backend 环境变量

```text
ASPNETCORE_ENVIRONMENT=Development
ConnectionStrings__Default=Server=localhost;Database=GrowthAI;Trusted_Connection=True;TrustServerCertificate=True;
Jwt__SigningKey=local-dev-signing-key
AiProviders__DeepSeek__ApiKey=
AiProviders__OpenAI__ApiKey=
AiProviders__Qwen__ApiKey=
```

## Frontend 环境变量

```text
VITE_API_BASE_URL=http://localhost:5000
VITE_APP_NAME=GrowthAI Lead Engine
```

## 本地开发依赖

- .NET 9 SDK
- Node.js LTS
- SQL Server Developer / Docker SQL Server
- Redis，可在早期阶段暂不启动
