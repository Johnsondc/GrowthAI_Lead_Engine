# 开发任务拆解

## Backend

1. 创建解决方案 `GrowthAI.sln`。
2. 创建项目：Api、Application、Domain、Infrastructure、Tests。
3. 定义实体：Tenant、AppUser、LeadCustomer、LeadSource、AiContent、AiTask。
4. 实现 EF Core DbContext 与迁移。
5. 实现客户池 CRUD、状态修改、负责人分配。
6. 实现 AI 内容生成接口与任务记录。
7. 实现数据分析聚合接口。

## Frontend

1. 创建 Vue3 + TypeScript + Element Plus 项目。
2. 完成路由：Dashboard、LeadPool、LeadDetail、ContentCenter、LandingManager、Analytics、Settings。
3. 完成 API Client 和类型定义。
4. 完成客户池列表和筛选。
5. 完成 AI 内容生成表单。
6. 完成 Landing Page 表单预览。
7. 完成数据分析卡片和图表。

## Database

1. 执行 MVP 建表脚本。
2. 插入租户、用户和线索来源种子数据。
3. 为 TenantId、Status、Platform、CreatedAt 建索引。
