# GrowthAI Lead Engine 文档总览

本目录是项目的产品与工程知识库，目标是让 Cursor、Qoder 或其他 AI 编码工具可以直接理解项目边界并继续开发。

## 阅读顺序

1. `00_Project/01_项目定位.md`：先理解为什么做、给谁做、第一版边界。
2. `00_Project/02_MVP目标.md`：明确 V1 只做获客闭环。
3. `01_Competitive/01_视频拆解.md`：理解参考产品的核心价值。
4. `02_PRD/01_MVP_PRD.md`：进入产品功能设计。
5. `03_Architecture/01_数据库设计.md`：理解数据结构。
6. `03_Architecture/02_API设计.md`：理解前后端契约。
7. `04_Development/01_开发任务拆解.md`：按任务进入编码。

## 文档分类

- `00_Project`：项目定位、目标、路线图、商业模式、行业版本。
- `01_Competitive`：视频拆解、竞品矩阵、页面矩阵。
- `02_PRD`：MVP PRD、用户流程、页面设计、字段字典、验收标准。
- `03_Architecture`：数据库、API、服务模块、集成、权限安全、部署。
- `04_Development`：任务拆解、开发 Prompt、里程碑、测试策略、环境配置。

- `05_Audit`：生成审查报告、运行限制、功能遗漏和业务逻辑对齐检查。
