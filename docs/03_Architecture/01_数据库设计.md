# 数据库设计

## MVP 六张核心表

1. Tenant：企业租户。
2. AppUser：员工账号。
3. LeadCustomer：客户线索。
4. LeadSource：线索来源。
5. AiContent：AI 生成内容。
6. AiTask：AI 任务记录。

## 状态枚举

LeadStatus：New、Contacted、Communicating、HighIntent、Won、Invalid。

SourcePlatform：Douyin、Xiaohongshu、Wechat、VideoAccount、OfficialAccount、Moments、QrCode、H5Form、MiniProgram。
