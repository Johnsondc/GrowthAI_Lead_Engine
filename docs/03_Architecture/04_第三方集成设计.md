# 第三方集成设计

## AI Provider

第一阶段只需抽象统一接口：

```text
IAiTextGenerator.GenerateAsync(request)
```

Provider 可包括 DeepSeek、OpenAI、Qwen。业务层不直接依赖具体厂商。

## 短信和电话

V1 可不接入。后续可接入短信提醒、外呼系统或企业微信。

## 新媒体平台

V1 不做自动发布和自动私信，只记录来源平台、账号和来源码。后续再接抖音、小红书、视频号开放接口。

## 二维码

二维码本质是指向 Landing Page 的来源码 URL。系统需能生成不同渠道码，例如：

```text
https://lead.growthai.com/l/maternity-cd-douyin-001
```
