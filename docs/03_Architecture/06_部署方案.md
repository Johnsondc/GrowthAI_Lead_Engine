# 部署方案

## MVP 单机部署

- Windows Server 或 Linux Server。
- ASP.NET Core API 作为后端服务。
- Vue 前端构建为静态文件，通过 Nginx 或 IIS 托管。
- SQL Server 作为主数据库。
- Redis 和 Hangfire 可在 v0.4 后接入。

## 推荐环境

- Dev：本地 SQL Server Developer + 本地 API + 本地前端。
- Staging：独立测试服务器和测试数据库。
- Production：应用服务器 + 数据库服务器分离。

## 配置项

- ConnectionStrings:Default
- AiProviders:DeepSeek:ApiKey
- AiProviders:OpenAI:ApiKey
- AiProviders:Qwen:ApiKey
- Jwt:Issuer
- Jwt:Audience
- Jwt:SigningKey
