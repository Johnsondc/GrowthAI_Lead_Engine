# 字段字典

## LeadCustomer

| 字段 | 中文 | 必填 | 说明 |
| --- | --- | --- | --- |
| Id | 客户 ID | 是 | 系统生成 |
| TenantId | 企业 ID | 是 | 多租户隔离 |
| Name | 客户姓名 | 是 | H5 表单或销售录入 |
| Phone | 手机号 | 否 | 用于电话跟进 |
| Wechat | 微信号 | 否 | 用于私域承接 |
| City | 城市 | 否 | 本地生活行业关键字段 |
| SourcePlatform | 来源平台 | 是 | 抖音、小红书、视频号、微信、H5、二维码 |
| SourceAccount | 来源账号 | 否 | 具体运营账号 |
| ConsultationContent | 咨询内容 | 否 | 用户原始问题 |
| InterestedProduct | 意向产品 | 否 | 月嫂、产康、课程等 |
| Status | 客户状态 | 是 | 新客户、已联系、沟通中、高意向、成交、无效 |
| OwnerUserId | 负责人 | 否 | 销售或运营人员 |
| Remark | 备注 | 否 | 跟进补充信息 |
| CreatedAt | 创建时间 | 是 | 入库时间 |
| UpdatedAt | 更新时间 | 是 | 最近修改时间 |

## AiContent

| 字段 | 中文 | 说明 |
| --- | --- | --- |
| Platform | 内容平台 | 抖音、小红书、视频号、朋友圈 |
| Industry | 行业 | 母婴、家政、教培等 |
| ProductName | 产品 | 月嫂、保洁、课程等 |
| TargetAudience | 目标客户 | 孕妇家庭、新手妈妈等 |
| Title | 标题 | 用于平台发布 |
| Body | 正文 | 小红书或朋友圈正文 |
| Script | 脚本 | 短视频脚本 |
| Tags | 标签 | 平台话题标签 |
| CallToAction | 引导动作 | 评论关键词、扫码、填表 |
