# MVP PRD

## 产品名称

GrowthAI Lead Engine

## 模块 1：企业管理

- 企业信息：名称、行业、城市、联系人。
- 员工：姓名、手机号、角色、状态。
- 角色：管理员、运营、销售、只读老板。

## 模块 2：客户池

- 客户列表：姓名、手机号、微信、城市、来源平台、来源账号、意向产品、咨询内容、状态、负责人、创建时间。
- 客户状态：新客户、已联系、沟通中、高意向、成交、无效。
- 客户详情：基本信息、来源信息、咨询内容、备注、跟进记录。
- 操作：新增、编辑、分配、修改状态、添加备注、导出。

## 模块 3：内容中心

输入：行业、产品、城市、目标客户、卖点、语气。

输出：

- 抖音：标题、视频脚本、开头 3 秒、评论引导、私信回复话术。
- 小红书：标题、正文、标签、封面建议、评论区引流话术。
- 视频号/朋友圈：短文案、转化话术、二维码引导文案。

## 模块 4：Landing Page

- 表单字段：姓名、电话、微信、城市、需求、意向产品、备注。
- 支持每个平台或账号生成独立来源码。
- 提交后写入客户池。
- 支持成功页提示添加微信或等待顾问联系。

## 模块 5：数据分析

- 今日新增客户。
- 按平台统计：抖音、小红书、视频号、微信、H5、二维码。
- 有效客户数量。
- 高意向客户数量。
- 成交客户数量。
- 来源转化率。
