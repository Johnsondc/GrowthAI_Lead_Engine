# 用户流程

## 运营流程

登录后台 → 进入内容中心 → 输入行业、产品、城市、目标客户 → AI 生成内容 → 复制发布到抖音/小红书/视频号 → 配置评论引导和二维码 → 等待客户咨询。

## 客户流程

看到内容 → 评论关键词或扫码 → 进入 H5 引流页 → 填写姓名、电话、微信、城市和需求 → 提交成功 → 进入企业客户池。

## 销售流程

登录后台 → 查看客户池 → 筛选新客户/高意向 → 领取或被分配客户 → 电话/微信联系 → 修改状态 → 添加跟进备注 → 成交或标记无效。

## 老板流程

登录后台 → 查看数据分析 → 了解今日新增、各平台来源、有效客户、高意向客户、成交客户 → 决定加大哪个平台和内容方向投入。
