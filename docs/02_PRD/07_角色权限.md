# 角色权限

| 能力 | 管理员 | 老板只读 | 运营 | 销售 |
| --- | --- | --- | --- | --- |
| 查看仪表盘 | 是 | 是 | 是 | 仅个人 |
| 管理员工 | 是 | 否 | 否 | 否 |
| 查看全部客户 | 是 | 是 | 是 | 否 |
| 查看自己客户 | 是 | 是 | 是 | 是 |
| 分配客户 | 是 | 否 | 是 | 否 |
| 修改客户状态 | 是 | 否 | 是 | 是 |
| 生成 AI 内容 | 是 | 否 | 是 | 否 |
| 管理引流页 | 是 | 否 | 是 | 否 |
| 导出客户 | 是 | 是 | 可配置 | 否 |

## 权限原则

默认所有数据按 TenantId 隔离。销售默认只看自己负责的客户，管理员和运营可看全量客户。
