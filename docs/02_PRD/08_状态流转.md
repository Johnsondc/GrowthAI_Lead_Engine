# 客户状态流转

## 状态定义

```text
新客户 → 已联系 → 沟通中 → 高意向 → 成交
                         ↓
                       无效
```

## 规则

- 新客户：通过表单、二维码、导入或手动新增进入系统。
- 已联系：销售已经电话、微信或私信触达。
- 沟通中：客户有明确需求，但尚未形成强购买意向。
- 高意向：客户有预算、时间、需求中的至少两个明确条件。
- 成交：客户完成购买或支付定金。
- 无效：空号、重复、非目标客户、无法联系或明确拒绝。

## 自动化建议

- 新客户超过 2 小时未分配，提醒管理员。
- 高意向客户超过 24 小时未跟进，提醒负责人。
- 无效客户必须选择无效原因。
