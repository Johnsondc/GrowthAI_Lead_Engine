# 页面设计

## 后台页面

1. 登录页：手机号/账号登录。
2. 首页仪表盘：今日新增、来源占比、有效客户、高意向客户。
3. 客户池：列表、筛选、状态、负责人、导出。
4. 客户详情：联系方式、来源、需求、备注、跟进记录。
5. 内容中心：AI 内容生成表单与结果区。
6. 引流页管理：表单模板、二维码、来源码。
7. 数据分析：平台趋势、客户状态、转化漏斗。
8. 企业设置：企业资料、员工、角色。

## H5 页面

- 标题区：行业化转化标题。
- 卖点区：3-5 个核心利益点。
- 表单区：姓名、电话、微信、城市、需求。
- 信任区：服务案例、证书、承诺。
- 提交成功页：提示顾问联系或添加微信。
