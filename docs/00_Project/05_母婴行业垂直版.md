# 母婴行业垂直版

## 场景

母婴行业适合作为第一垂直场景，因为月嫂、育儿嫂、产康、催乳、早教等服务高度依赖咨询和信任建立，且用户会主动留下电话或微信。

## 典型客户字段

- 姓名
- 电话
- 微信
- 城市
- 预产期 / 宝宝月龄
- 需求类型：月嫂、育儿嫂、产康、催乳、早教
- 预算区间
- 咨询内容
- 来源平台
- 来源账号
- 跟进状态

## 内容模板方向

- 孕晚期准备清单
- 月嫂避坑指南
- 产后恢复时间表
- 新手妈妈常见问题
- 成都/本地月嫂价格说明
- 真实服务案例

## Landing Page 字段建议

母婴版本 Landing Page 必须收集预产期或宝宝月龄，以便销售判断跟进优先级。
