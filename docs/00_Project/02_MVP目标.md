# MVP 目标

## 唯一目标

持续帮助企业获取客户线索，并让线索可以被统计、分配和跟进。

## MVP 做什么

- 客户池：沉淀所有新媒体渠道来的客户。
- 内容中心：AI 生成抖音、小红书、视频号等平台内容。
- 引流页：承接评论、私信、二维码和 H5 表单。
- 客户管理：记录客户状态、负责人、备注和需求。
- 数据分析：统计新增客户、来源平台、有效客户、高意向客户。

## MVP 不做什么

- 不做订单、支付、售后、库存、复杂财务。
- 不做自动发布和全平台私信机器人。
- 不做 AI 数字人、RAG、复杂 Agent 编排。
- 不做过重的销售 CRM 流程。

## 第一阶段 KPI

- 每天新增线索 10+。
- 每周新增客户 70+。
- 来源平台可追踪。
- 有效客户和高意向客户可标记。
- 每条线索都能看到负责人和跟进状态。
