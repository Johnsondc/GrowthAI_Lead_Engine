# 商业模式

## 目标客户

优先服务本地生活和服务型企业，例如母婴、家政、教培、医美、房产中介。这些行业的共同点是客单价较高、依赖咨询转化、客户线索有明确生命周期。

## 收费方式

### SaaS 订阅

- Starter：单企业、3 个员工、基础客户池和内容生成。
- Growth：10 个员工、多来源码、数据分析和 AI 模板。
- Pro：多门店、多账号、团队权限、导出和集成能力。

### 增值收费

- AI 生成额度包。
- 行业内容模板包。
- Landing Page 高级模板。
- 私域运营和代运营服务。

## 关键商业指标

- 新增企业数。
- 激活企业数。
- 每企业月新增线索数。
- 每线索获客成本。
- AI 内容使用次数。
- Landing Page 提交转化率。
- 付费转化率与续费率。
